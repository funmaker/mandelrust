- [X] Smooth coloring
- [X] Fixed point math library
- [X] Inline assembler code for optimalized multiplication and addition
- [ ] Dynamic precision depending on zoom level
- [ ] Interface for changing color gradient
- [ ] High resolution shots
- [ ] Recording
- [ ] Different fractals
